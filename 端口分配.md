# 端口分配方案

## 端口规划表

### 后端 API 端口

| 环境 | 端口 | 访问地址 | 说明 |
|------|------|----------|------|
| dev | 8118 | http://localhost:8118 | 后端开发环境 |
| staging | 8119 | http://localhost:8119 | 后端预发布环境 |
| prod | 8120 | http://localhost:8120 | 后端生产环境 |

### 数据库端口

| 环境 | MySQL 端口 | Redis 端口 | RustFS 端口 |
|------|------------|------------|-------------|
| dev | 3307 | 6383 | 8121 |
| staging | 3308 | 6384 | 8122 |
| prod | 3309 | 6382 | 8123 |

### 数据库连接信息

| 环境 | 数据库名 | 用户名 | 密码 |
|------|----------|--------|------|
| dev | together_dev | root | root123 |
| staging | together_staging | root | root123 |

## 端口分配规则

- **API 端口**: 8118-8120 (dev/staging/prod)
- **MySQL**: 3307-3309
- **Redis**: 6382-6384
- **RustFS**: 8121-8123

## 宿主机访问

由于 Jenkins 运行在 Docker 容器内，服务访问宿主机服务使用:
- `host.docker.internal` (容器内访问宿主机)
- `localhost` (宿主机访问容器映射端口)
